Overview

Aplikasi desktop fotobooth profesional yang dirancang untuk kebutuhan event seperti pernikahan, ulang tahun, wisuda, dan acara korporat.

Modern & Elegan

Dibangun dengan Python + PyQt6, aplikasi ini menawarkan antarmuka yang modern dengan tema gelap dan aksen emas yang premium.

Dark Theme

Nyaman digunakan di venue dengan pencahayaan rendah

Gold Accent

Tampilan premium dan profesional untuk branding

Fitur Utama

Semua yang Anda butuhkan untuk menjalankan fotobooth profesional dalam satu aplikasi terintegrasi.

Multi-Camera Support

Integrasi dengan Canon DSLR via digiCamControl dan dukungan webcam sebagai fallback.

Template System

Terapkan frame template PNG dengan transparansi pada foto secara otomatis.

A4 Canvas Editor

Susun hingga 5 foto dalam layout A4 dengan drag-and-drop yang intuitif.

Google Drive Integration

Upload otomatis ke Google Drive dengan autentikasi OAuth2 yang aman.

QR Code Generator

Generate QR code untuk link download, tamu bisa unduh foto langsung dari HP.

Print Ready Export

Export langsung ke format A4 yang siap cetak dengan kualitas tinggi.

Contoh Hasil Output

Template dapat dikustomisasi sesuai tema event. Berikut contoh output dengan frame template custom.

Custom Frame Design

Template dikustomisasi sesuai tema event. Contoh: tema "KOMIK 2025" dengan layout photostrip 3 foto.

Transparent PNG Support

Mendukung template PNG dengan transparansi untuk overlay yang sempurna.

High Resolution

Output berkualitas tinggi, siap untuk cetak maupun share digital.
